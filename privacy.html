1. Overview

This Privacy Policy describes how Geekfeed Inc. ("we", "our", or "us") handles data in connection with the YouWire MCP Server — a Model Context Protocol server that enables Claude to access the YouWire compliance call recording platform.

YouWire MCP acts as a secure bridge between Claude and your organization's YouWire account. We are committed to protecting your privacy and handling data responsibly in accordance with Japan's Act on the Protection of Personal Information (APPI).

2. Data the MCP Server Accesses

When you connect Claude to YouWire MCP, the server may access the following data from your YouWire account, only when explicitly requested by you:

Call Recording Metadata

  • Date, time, and duration of calls
  • Call direction (incoming / outgoing / meeting / web meeting / upload)
  • Recording source (mobile, landline, Teams, web conferencing, in-person)
  • Phone numbers of local and remote parties
  • Display names of local and remote parties
  • Group name and user name associated with the recording
  • Protection status flag
  • Download datetime (if the recording has been downloaded)
  • File size of uploaded recordings
  • Record creation and update timestamps

Location Data

  • Latitude and longitude coordinates associated with mobile recordings (when available)
  • Location string (when available)

Transcription Data

  • Speech recognition transcripts of call recordings (when speech recognition is enabled)
  • Timestamped transcript segments with speaker direction indicators

Compliance Tags

  • Tag titles, content text, category names and IDs
  • Tag creation and update timestamps

User Account Information

  • Username and user ID
  • Permission level and access scope (operating range)
  • Group memberships
  • Feature permissions (search, download, delete, protect, tag)

Aggregated Statistics

  • Call counts, total and average durations, broken down by direction

The MCP server does not access raw audio files directly. Transcripts are generated by YouWire's speech recognition engine and stored within YouWire's infrastructure.

3. How Data is Used

Data accessed through YouWire MCP is used solely to:

  • Respond to your queries within the Claude conversation
  • Return search results, transcripts, and statistics as requested
  • Perform write operations (tag management) that you explicitly initiate

We do not use your call recording data to train AI models, for advertising, or for any purpose beyond fulfilling your immediate request.

4. Data Storage and Retention

MCP Server (Session Data)

The YouWire MCP server does not persistently store your call recording data. Data retrieved during a session is held in memory only for the duration of the API request and is not written to disk or retained after the response is sent.

Authentication

YouWire MCP supports the following authentication methods:

  • OAuth 2.1 + PKCE (recommended): OAuth access tokens and refresh tokens are stored securely by Anthropic's infrastructure as part of the connector authentication flow. These are encrypted at rest and used solely to authenticate your requests to the YouWire MCP server.
  • API key / session token: For non-OAuth connections (e.g., Claude Desktop direct configuration), a session token may be used. This token is managed within your client configuration and is not stored by the MCP server itself.
  • OAuth Access tokens expire after 1 hour (auto-refreshed)
  • OAuth Refresh tokens expire after 30 days
  • Tokens are invalidated when you disconnect the connector in Claude settings

YouWire Platform Data

Call recordings, transcripts, and tags remain stored within YouWire's infrastructure under your organization's account. Retention policies are governed by your YouWire service agreement and applicable regulations.

5. Data Transmission Security

  • All communication between Claude and YouWire MCP is encrypted using HTTPS / TLS 1.3
  • OAuth authentication uses OAuth 2.1 + PKCE — your YouWire password is never shared with Anthropic or Claude
  • Access is scoped per-user based on YouWire's operating_range permission model — the MCP server cannot access data beyond your authorized scope
  • Security events and access logs are automatically recorded within YouWire's infrastructure

6. Data Sharing

We do not sell, rent, or share your data with third parties except:

  • Anthropic: As part of the MCP connector infrastructure, request/response data passes through Anthropic's servers. Please refer to Anthropic's Privacy Policy for details.
  • Legal obligations: If required by applicable law or regulation.

7. Your Rights (APPI)

Under Japan's Act on the Protection of Personal Information, you have the right to:

  • Request disclosure of personal information we hold about you
  • Request correction of inaccurate personal information
  • Request deletion of your personal information
  • Request suspension of use of your personal information

8. Geographic Availability

YouWire MCP is currently available to organizations based in Japan only. Data is processed and stored within infrastructure compliant with Japanese data protection laws.
